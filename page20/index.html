记录：劳动节这三五日

今天第三天了，稍微有点儿累。时间高度密集，大量的词汇涌过来，有点儿应接不暇的感觉。
正巧今天晚饭后不小心把电脑弄出个小毛病，于是耽误到快八点才出宿舍，在教七找了个角落。
开始看了几个词，实在不在状态，跟这个假期的天气一般，躁动。
于是，换教室，开电脑，松弛一下紧绷了这么久（久？……）的神经。
回顾一下子前几天的行踪哦。

貌似在周五晚上帮人弄得电脑并没当场好，而且回去之后又出毛病了，于是立刻马上迅速回头抱回来两台（前一天修的一台也出问题了，不过根据后来检查，责任不在我）。那个晚上就过去了。

周六干什么了？貌似肯定是电影节的闭幕式！早上跑到雍和宫，看了那里的佛像和建筑。回到学校的时候碰到吁吁跑过来，有电影节的名额，于是决定跟踪他们去吧。把约好的修电脑业务推迟了。
晚上的电影节颁奖仪式果然又没让我失望。影星来了不少，不过总感觉红地毯上的还不够多……也许所有观众都会有点儿不知足吧，希望更多才好。在外面留了张照片，可惜拍的效果一般──谁让我没出过什么好照片呢。
进场的时候，带着雯雯小妞，又一次不小心跑到了台前……也许拿相机坐在后上方的各位极度抓狂，但这个世界就是这样，没什么是公平的。

周日，起个大早，帮人杀毒未果，重装搞定。
还惊奇的发现，新浪blog刚刚弄出来的开源软件专题里竟然收录我的那篇文章，其实只是一个课堂作业而已。直接后果就是在google里输入我的姓名，就可以在第一页上看到我那篇文章了。新浪果然强大，而社会舞台上一出出闹剧，也通过奇怪的方式影响到我。

周一，两节课都没去上，颓废啊。

周二了，正是五一劳动节，本应和新东方的朋友提前见面打个招呼，因为第二天就开始了，却再次错过──放假了，到场都是一个乱啊。
半下午抽风，带个小孩儿跑到天安门。看了降旗，仍然是乱，并且工交车下满是饥渴的目光，恐惧驱使我们往前走；冷了，走到西单，在那里吃冰激凌。出来仍然冷。坐车回来9点了，饿，直接到北邮南门小巷的清真去吃手抓饭。俩小时才填饱……汗死俩瘦人大半夜吃那么多。

周三开始，就每天到新东方了。
第一天约好时间，同伴却在凌晨5点的时候发来短信告知她通宵了……自己先去了，刚上车却又收到她已出门的消息。
第一感觉不值得疑问，就是真多真快。到了第二天仍然同样的感觉，不过留笔记的时候多费点儿心，于是复习的时候方便多了。我不是个称职的学习伙伴，因为这门课程让我感到太吃力了，真的很对不起这热情提问我单词的同伴。

3天了，回来我向老二炫耀，因为他对我的蔑视已经不攻自破了，我坚持了下来。
后面还有一半儿的课，加油加油。我一定行的

今天早上，一年级的小孩子们已经出发到军训的基地了。昨晚还真把他们忙坏了。跑过去看了几个，简单鼓励一下。他们不会为这几天的生活后悔的。

受微软反盗版升级补丁困扰的朋友请看

受微软反盗版升级补丁困扰的朋友请看：

注册表:HKEY_LOCAL_MACHINE\SOFTWARE\microsoft\Windows NT\ CurrentVersion\Winlogon\Notify\WgaLogon，删除它，重新启动(一定要重起要不然删不了C盘的文件)， wgatray.exe就不会自动启动了，再把C:\windows\system 32中的wgatray.exe删除，再搜索一下，在另外的文件夹下还有，统统删除，重起.然后再进HKEY_LOCAL_MACHINE\ SOFTWARE\microsoft\Windows NT\ CurrentVersion\Winlogon\Notify\WgaLogon，删除它终于OK了,下次记得不要再更新.

什么被他弄丢了

什么被他弄丢了
 
他加入这个班的时候，高考的倒计时牌子上是个两位数。
让人窒息的空气中，偶尔有人放肆宣泄着青春。
大概还是春天，像今天一样，没有什么能够压制这个季节赋予万物的生命力。
他总是专注于自己那几本很老却很整齐的书。前面，他的路在何方，没去想，太遥远的事情没有意义，尽管这个距离不太远。
每天结束了晚自习，总有几个行动缓慢的家伙，讨论着无聊的话题。模拟考试，大学，等等等等……他们的视界的全部。
他不喜欢提问老师，因为，不讷言语的他，总不能清楚表达自己的问题。于是，左邻右舍的老师成了真正的朋友。
梧桐花开了，又落了；
槐花开了，又谢了；
路边的月季花，也开了又谢了。
然后，高考开始了，接着又结束了。
夏天，正在北京照料四叔，懒惰的他，从地图上随手圈了一个大学，填在了志愿表上，他有自信的理由。
和他一起来北京的，有一位女孩曾作为左邻右舍的老师。她很好，很好……此时他相信，语言描述将是如此多余。
他们一起逛街一起自习一起坐在长凳子上发呆，一起等待晚上最后一班公交车。50分钟的地铁或者50分钟的电话成了每天的必修课。
又过了一年，又过了一年。
……
桃花开了有谢了；
梧桐花开了，又落了；
槐花开了，又谢了；
路边的月季花，也开了又谢了。
很像是做了个梦，他经历了最丰富的大学生活，她也经历了最丰富的大学生活。他和她已经有一年没有在一起了，甚至没有通过一次电话。倔强的他们，面对着一个无奈。
想起从前，一起逛街吃东西；想起从前，一起从五环步行到王府井去……现在，什么都没有了什么被他们弄丢了？什么被他弄丢了？

关于由递推公式求数列的通项

方法1、
配方法。
配方后换元，减少递推公式中项的数目，简化运算。
a[n] = 2a[n-1] - a[n-2] 
a[n] - a[n-1] = a[n-1] - a[n-2]
设b[n] = a[n] - a[n-1]
则b[n] = b[n-1]，为等差数列。然后可以根据初始条件进行计算。
 
方法2、
特征方程。
 
方法3、
生成函数。
A = a[0] + a[1]x + a[2]x^2 + … + a[n]x^n + ...
kxA = ...
(1 + kx)A = a[0] + { a[2] + a[1]k }x + ...
若一直a[n] + a[n-1]k = 0，则后面所有项为0。
可求得A = a[0] / (1 + kx )。
a[1] = A'(x)/1!, a[2] = A''(x)/2!, ......
可求出其通项a[n] = A(x)^(n) / n!
