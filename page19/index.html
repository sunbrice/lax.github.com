关于由递推公式求数列的通项

方法1、
配方法。
配方后换元，减少递推公式中项的数目，简化运算。
a[n] = 2a[n-1] - a[n-2] 
a[n] - a[n-1] = a[n-1] - a[n-2]
设b[n] = a[n] - a[n-1]
则b[n] = b[n-1]，为等差数列。然后可以根据初始条件进行计算。
 
方法2、
特征方程。
 
方法3、
生成函数。
A = a[0] + a[1]x + a[2]x^2 + … + a[n]x^n + ...
kxA = ...
(1 + kx)A = a[0] + { a[2] + a[1]k }x + ...
若一直a[n] + a[n-1]k = 0，则后面所有项为0。
可求得A = a[0] / (1 + kx )。
a[1] = A'(x)/1!, a[2] = A''(x)/2!, ......
可求出其通项a[n] = A(x)^(n) / n!

我们无处发泄的自由 --《梦想照进现实》观后

我们无处发泄的自由
《梦想照进现实》观后

观看时间：2007年04月18日晚八点；地点：中国电影资料馆
徐静蕾，韩童生

太极拳，拳打卧牛之地。太极拳的美妙，就在一招一式中，蕴含着朴实的哲学。不知道老徐是不是也打太极拳。
她的这个电影，就在狭小的小屋子里，展现了一丝丝浮躁的思想。正是王朔作品一向要表达的东西。这家伙，真还有点儿前瞻意识，那么多年前就写的这一套，竟被大家渐渐接受了。
夜，有点儿安静。走廊里，敲门声。打火机被转来转去，掉在地上的声音。屋子里，沙发，茶几上都是如我们宿舍一般的凌乱。大家都在忙着做事，没人会闲下来，整理一下。
社会是一台很大的机器，没有谁不被牵着走来走去。当大众的意识进入我们的大脑，个人甚至无法记起自己曾说过的话，甚至连听到的音乐都能够被虚构出来。谁也不知道自己干了什么。很多人要自由，“想干嘛干嘛”。但是又发现自己还是钻进了预先摆好的模子里。
于是，无尽的扯谈，绕来绕去，依然无头苍蝇般，从不同的视角，用不同的方式，搜索这个答案。但是，谁找到了呢？
最后，置旁人狂敲门于不顾，大扮鬼脸，肆意展示藏在内心的压抑，终究是不能摆脱掉，这个乱了套的世界。

关于盗版软件的前世今生 ──商业软件与开源软件的思索

关于盗版软件的前世今生
──商业软件与开源软件的思索

作者：刘兰涛（liulantao@liulantao.com）


一、盗版产生的背景：
1、技术的进步和更新，吸引了很多倡导自由，以破解为乐趣的个人和组织。每一次新产品的发布，都是这些破解组织展示技术的好机会。如果一个组织能够在最短时间内把一个软件破解，将成为该软件的唯一破解版本，而在各大破解软件共享社区流行，除非有明显优于这一破解的版本将其取代。在这种破解竞赛中，各种工具和技术发展成熟，产生出大量被破解的商业软件。
2、由于计算机的普及，渗透入人们生活的方方面面。用户使用软件，产生了大量的需求。在没有条件购买正版软件的地区，盗版软件大有市场。盗版商追求利润，利用cracker的成果，进行批量生产，攫取暴利。某些经济较为落后地区，则因为难以承担高额的费用，而大规模使用盗版软件，以推进信息化，这在经济发展过程中是必然要经过的阶段。
3、软件商的策略；以微软在中国市场推广windows为例，当其windows 98 操作系统将在中国市场推出的时候，通过放任盗版，培养起用户依赖性和忠诚度，同时挤压竞争对手以攫取市场占有量。一旦成功，则通过打击盗版，迫使政府部门和大型企业使用其正版系统。这是对我们的政治经济文化很具有威胁性的策略，应该予以警惕。

二、软件的分类：
1、商业软件；商业软件（commercial software）是在计算机软件中，指被作为商品进行交易的软件。软件的商业化，对于很多软件公司的生存和发展具有很重要的意义。为商业公司，为投资商产生利润的来源。
2、开源软件(Open Source Software)；又称自由软件(Free Software)，它专指源代码公开的软件，不一定是免费，但用户可以通过源代码来了解软件如何运行，并能够在出现错误时自行解决。商业软件一般不会公开其源代码，这样，如果出现问题，只有通过其售后服务来解决，周期将变得非常漫长。
3、按应用分类：系统软件，应用软件等。盗版软件中，系统软件的盗版率比较高，因为每台电脑都要有其操作系统才能够运行，普通用户很可能无法承担高额的费用而使用盗版。几年前我国的法律制度仍然允许个人用户和非赢利组织以研究为目的使用破解软件。

三、使用盗版软件要三思：
1、用户权利无保障；售后的服务与支持，作为软件使用者要考虑的一个大问题，难以被忽略。大部分软件公司提供正版软件的升级和售后服务，而盗版软件则无法提供。当出现问题的时候，盗版软件使用者只能独自承担一切损失。
2、侵犯软件生产商的权利；这点毫无疑问，软件商的利润被盗版商所瓜分，软件制造者不能获得应有的收益，给软件的进一步发展造成经济上的困难。
3、对法律制度形成挑战；我国已经出台相关的法律和制度对软件进行保护，使用盗版是对法律制度的侵犯。

四、解决思路：
1、发展商业软件，形成良性竞争来规范市场；将竞争机制引入软件业，防止垄断，从而提高生产率，降低成本。
2、授权制度的改变；由单一的授权形式转换为根据用户性质进行授权，个人用户和商业用户，大型企业和小企业区别来对待。这需要软件公司与客户们进行深入的探讨。BSD系统就采用了其特有的授权方式。
3、免费软件；功能和服务受到限制，可以实现某些方面的功能。某些免费软件已经成功找到赢利模式，比如qq，msn，迅雷等，通过附加的业务，实现了赢利。
4、开源软件；体现了对用户知情权的尊重，使用户了解软件的功能。很多开源软件是免费的，开发者遵循自由开放的原则，支持最新的通用标准。比如开源的GNU/Linux操作系统，BSD操作系统等，当然某些Unix操作系统开源但不是免费的。openoffice.org的办公套件，mozilla公司的产品，都已经成为有影响力的软件。SUN公司也已经宣布对其Java实行开源政策，开源软件更加有竞争力。多数情况下，开源软件是值得公司和个人采用的好方案。在开源软件的影响下，微软已经越来越意识到标准化的作用，在office 2007中使用了新的文件格式，并且力主使其成为开放的通用文件格式。

可以预见，在不久的将来，更成熟的法律制度建立之后，盗版软件的市场就难以存在。而软件业将会在自由软件的促进下，种类更加丰富多彩，技术进步更加迅速。


注：
本文使用OpenOffice.org Writer在Fedora GNU/Linux操作系统上完成。
Fedora GNU/Linux为Red Hat公司产品；OpenOffice.org为Sun Microsystems Inc.产品。
