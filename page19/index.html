仍然只能错过的交点

早上又跑到宿舍去，帮那个女孩儿弄个软件。
一年过去了，看起来还是那么舒服的女孩儿。
记得第一次见她的时候，她情绪不大好，
也许源于那个季节酝酿的情绪，
让很多人做出不理智的行为，更加影响了尚在萌芽的思绪。
于是郁闷横行，大家彻夜不眠，通宵叫嚣，大肆吃喝玩乐。
来自对面的认同也艰难如斯，日子终于开始漫长。
太多的偶然给人虚幻的希望，思想逐渐变得幼稚。
但是，总有必然的无奈和无聊，
走过路过最终仍然要错过。

My Months Experience On Fedora.

I can still remember the time, when I spent a whole day download
Fedora Core 5 install images from redhat official website.
I have tried several Linux versions on my Dell laptop. The original
installed Microsoft Windows XP make no means to me as I can't arrange
many system services on it. So I find several Linux distributions like
Mandrake, Suse, in the FTP site of campus, and burn them into CDs. But
I think they are too old not because there relatively long history,
but there missing function: non of them can drive my network device,
though I must say Suse shows an excellent appearance.
Then Fedora Cora 5 did it. I can connect to the internet and find
answers to various of problems and questions, that's fabulous. At that
time I was not familiar with Linux style enough to solve some of the
problems.
Soon Fedora Core 6 come out with the ability to recognize my sound
card. It is a WOW! experience because FC6 added several features which
make it an perfect Desktop environment. I learned linux style
opereation on FC6, server my computer as an mini http&ftp server, they
are all so exciting for a beginner as me.
Months later I have knowledge of Fedora Development project, which add
the newest packages to Fedora to form a new version at the end. I
joined the group and began the excellent experience.
I would say: Linux is funny, and Open Source softwares will do
everything perfect, better than most of other known operating systems.

给远方的朋友

最近好吗？一句最普通，最温暖的问候。找不出任何美妙的句子能比它温暖，也只好用上这古老的问句。愿你的一切都好……

去年的信，写到今年，终于没有成型。只有记忆的碎片，聚集起来，倒是把我博客的内容给填满了。
朋友的祝福，在心底，分量如此重，再不回复一下，恐怕我要出现心理问题了。所以，为了自己的健康，赶凑出这个信封里的东西吧。
本来谈谈自己的感受，会是那么惬意。却因为思维总是不受控制地发散，只好把记录的这些碎片整理出来，给你看看了。隔了好几年没有写东西，语言已经不能跟上思想，遣词造句起承转合的影子也没有出现，即使作为记录，也已不明晰了。随便看看吧。

香山。

5月13日，香山

中午到达山脚下，进入。由南侧向上登。渐行渐窄的山路，并没有多少麻烦。路过静翠湖，上到香山寺。
香山寺只剩下残余的基座和石屏风。从南侧小门出去，一块巨石耸立，使劲仰头，石巅有一亭子，旋绕而上，就到了。
此处是到达的第一个视角开阔的地点。向东，隐约有北京市区楼房的影子；向北，几个山头都在眼中，山坡树林茂盛，可以想见秋季漫山红叶时候的景色。再往山上，小路一段之后就隐藏在密密的山林里。
选的这条路是很偏僻，几乎都是很原始的，用木桩和碎石搭建的路。再几经盘旋，听到人声；又行几步，就见到人们从另外几条宽阔一点儿的路过来。回首向上，可以看见这里的山巅峰了。咬咬牙，很快就到了山顶。
躺在石墙下侧的石板上，看片片白云，和明蓝的天。心旷神怡之所在，让人再难有人生壮志或辗转缠绵之思绪。兴奋之下将此时感受炫耀式的短信给数位老友，尤其那正在几千里外奋战高考的某孩子：太善感悟的她在此地还会有什么感悟吗？
复前行，至香炉峰，只有两个平台供游人拍摄，索然无味，倒是可以将京城半侧收入眼下。
下山，经途有西山晴雪之处，忆及彼处断桥残雪，未尝真有雪片飘舞而下，却又有皎皎月光，映射漫山遍野的雪白。一定在月圆之夜才能体会到，因为古往今来，月圆留下的情感实在是太丰富了，唯有如此美景才能与之搭配。太白笔下"地上霜"，即此时。
有片林，遮盖四处的天空，静下心，听风声涛声。有束阳光，华丽的明亮的，沿山坡方向，透过重重树枝下来；越过大路，仍然华丽的明亮的，继续穿行，越过下方的小路，不见止息之处。我无心打扰她，也无法再为她的前途担忧，能在彼此的路途间设置一个交点，感谢造化了。
暮色将下，隔云钟处，处在浮云之后，听各处禅寺钟声传来，也是一个值得驻足的所在。

那时花开，那时花已谢了。

最早闻到的是梧桐花的香气，弥漫了整个院子的春天。
因为高中的时候，学校也有几棵高大的梧桐，
谁也说不清，那些梧桐，见证了多少青春美好的时光。

槐花的香气也非常浓烈，
想起了小学时候的那条路，
槐树开花季节，每次上学，都要爬到旁边的树上，
采下几朵来，放在嘴里嚼。
槐花是我吃过的花中最甜的了。

那时花开，
那时花已谢了。

记录：劳动节这三五日

今天第三天了，稍微有点儿累。时间高度密集，大量的词汇涌过来，有点儿应接不暇的感觉。
正巧今天晚饭后不小心把电脑弄出个小毛病，于是耽误到快八点才出宿舍，在教七找了个角落。
开始看了几个词，实在不在状态，跟这个假期的天气一般，躁动。
于是，换教室，开电脑，松弛一下紧绷了这么久（久？……）的神经。
回顾一下子前几天的行踪哦。

貌似在周五晚上帮人弄得电脑并没当场好，而且回去之后又出毛病了，于是立刻马上迅速回头抱回来两台（前一天修的一台也出问题了，不过根据后来检查，责任不在我）。那个晚上就过去了。

周六干什么了？貌似肯定是电影节的闭幕式！早上跑到雍和宫，看了那里的佛像和建筑。回到学校的时候碰到吁吁跑过来，有电影节的名额，于是决定跟踪他们去吧。把约好的修电脑业务推迟了。
晚上的电影节颁奖仪式果然又没让我失望。影星来了不少，不过总感觉红地毯上的还不够多……也许所有观众都会有点儿不知足吧，希望更多才好。在外面留了张照片，可惜拍的效果一般──谁让我没出过什么好照片呢。
进场的时候，带着雯雯小妞，又一次不小心跑到了台前……也许拿相机坐在后上方的各位极度抓狂，但这个世界就是这样，没什么是公平的。

周日，起个大早，帮人杀毒未果，重装搞定。
还惊奇的发现，新浪blog刚刚弄出来的开源软件专题里竟然收录我的那篇文章，其实只是一个课堂作业而已。直接后果就是在google里输入我的姓名，就可以在第一页上看到我那篇文章了。新浪果然强大，而社会舞台上一出出闹剧，也通过奇怪的方式影响到我。

周一，两节课都没去上，颓废啊。

周二了，正是五一劳动节，本应和新东方的朋友提前见面打个招呼，因为第二天就开始了，却再次错过──放假了，到场都是一个乱啊。
半下午抽风，带个小孩儿跑到天安门。看了降旗，仍然是乱，并且工交车下满是饥渴的目光，恐惧驱使我们往前走；冷了，走到西单，在那里吃冰激凌。出来仍然冷。坐车回来9点了，饿，直接到北邮南门小巷的清真去吃手抓饭。俩小时才填饱……汗死俩瘦人大半夜吃那么多。

周三开始，就每天到新东方了。
第一天约好时间，同伴却在凌晨5点的时候发来短信告知她通宵了……自己先去了，刚上车却又收到她已出门的消息。
第一感觉不值得疑问，就是真多真快。到了第二天仍然同样的感觉，不过留笔记的时候多费点儿心，于是复习的时候方便多了。我不是个称职的学习伙伴，因为这门课程让我感到太吃力了，真的很对不起这热情提问我单词的同伴。

3天了，回来我向老二炫耀，因为他对我的蔑视已经不攻自破了，我坚持了下来。
后面还有一半儿的课，加油加油。我一定行的

今天早上，一年级的小孩子们已经出发到军训的基地了。昨晚还真把他们忙坏了。跑过去看了几个，简单鼓励一下。他们不会为这几天的生活后悔的。
