受微软反盗版升级补丁困扰的朋友请看

受微软反盗版升级补丁困扰的朋友请看：

注册表:HKEY_LOCAL_MACHINE\SOFTWARE\microsoft\Windows NT\ CurrentVersion\Winlogon\Notify\WgaLogon，删除它，重新启动(一定要重起要不然删不了C盘的文件)， wgatray.exe就不会自动启动了，再把C:\windows\system 32中的wgatray.exe删除，再搜索一下，在另外的文件夹下还有，统统删除，重起.然后再进HKEY_LOCAL_MACHINE\ SOFTWARE\microsoft\Windows NT\ CurrentVersion\Winlogon\Notify\WgaLogon，删除它终于OK了,下次记得不要再更新.

关于由递推公式求数列的通项

方法1、
配方法。
配方后换元，减少递推公式中项的数目，简化运算。
a[n] = 2a[n-1] - a[n-2] 
a[n] - a[n-1] = a[n-1] - a[n-2]
设b[n] = a[n] - a[n-1]
则b[n] = b[n-1]，为等差数列。然后可以根据初始条件进行计算。
 
方法2、
特征方程。
 
方法3、
生成函数。
A = a[0] + a[1]x + a[2]x^2 + … + a[n]x^n + ...
kxA = ...
(1 + kx)A = a[0] + { a[2] + a[1]k }x + ...
若一直a[n] + a[n-1]k = 0，则后面所有项为0。
可求得A = a[0] / (1 + kx )。
a[1] = A'(x)/1!, a[2] = A''(x)/2!, ......
可求出其通项a[n] = A(x)^(n) / n!

我们无处发泄的自由 --《梦想照进现实》观后

我们无处发泄的自由
《梦想照进现实》观后

观看时间：2007年04月18日晚八点；地点：中国电影资料馆
徐静蕾，韩童生

太极拳，拳打卧牛之地。太极拳的美妙，就在一招一式中，蕴含着朴实的哲学。不知道老徐是不是也打太极拳。
她的这个电影，就在狭小的小屋子里，展现了一丝丝浮躁的思想。正是王朔作品一向要表达的东西。这家伙，真还有点儿前瞻意识，那么多年前就写的这一套，竟被大家渐渐接受了。
夜，有点儿安静。走廊里，敲门声。打火机被转来转去，掉在地上的声音。屋子里，沙发，茶几上都是如我们宿舍一般的凌乱。大家都在忙着做事，没人会闲下来，整理一下。
社会是一台很大的机器，没有谁不被牵着走来走去。当大众的意识进入我们的大脑，个人甚至无法记起自己曾说过的话，甚至连听到的音乐都能够被虚构出来。谁也不知道自己干了什么。很多人要自由，“想干嘛干嘛”。但是又发现自己还是钻进了预先摆好的模子里。
于是，无尽的扯谈，绕来绕去，依然无头苍蝇般，从不同的视角，用不同的方式，搜索这个答案。但是，谁找到了呢？
最后，置旁人狂敲门于不顾，大扮鬼脸，肆意展示藏在内心的压抑，终究是不能摆脱掉，这个乱了套的世界。
