Installing Chromium Web Browser on Fedora 11.

[lax@homex ~]$ sudo vim /etc/yum.repos.d/chromium.repo
[chromium]
name=Chromium Test Packages
baseurl=http://spot.fedorapeople.org/chromium/F$releasever/
enabled=0
gpgcheck=0
