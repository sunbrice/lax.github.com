--- layout: post name: windowsxpdcomlaunch title: WindowsXP连接无线网络必须开启DcomLaunch服务 time: 2008-05-05 20:38:00.000000000 +08:00 --- 如果要开机自动连接无线网络，必须设置下面的服务为自动启动：
Service name: DcomLaunch
Dispaly name: DCOM Server Process Launcher
Description: Provides launch functionality for DCOM services.
Path to executable: C:\WINDOWS\system32\svchost -k DcomLaunch

此服务可以保存以往的身份认证设置，如果不启动此服务，则无法完成无线网络身份认证。
