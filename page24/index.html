告别了一天


昨天。。是周四，现在已经是周五的凌晨了。
几个小时之后有一个很困难的考试，对于我来说，是很困难。
宣告生活的一种状态将要改变，用了一年的时间，探索着生活的一种新方式。
很有趣、刺激、疲惫、充实。
遭遇很多人和事件，面对无数辛酸感动和无奈。
通宵疯狂过，拼命干过活。
有了一个个辉煌的战绩，也走过了一年的光阴。
真的好有意义。

昨天下午推掉了班上发放补助的活儿。
这个活儿是老师偏心于我专门给你的，一直很感动我，
但是现在这个情况，只能放弃它了。

心情低落，没有了任何心境。
放松之后才能开始，我准备开始了。

在这个孤独的城市，我们是如此无助。
却不能发现，消失在夜幕中的那个影子
在游荡，在找寻来自远方的求助信号
他，从寂寞中诞生，爆发过，又回到平淡。
湮没了。依然生命顽强。
我已不如当初能遗忘干净。所以，
请你给我一点点消息，偶尔报下平安，好吗


听过很多朋友倾诉，也经历过自己的感情波折，
学会了遗忘。遗忘本来是最好的药剂，但是多年来积存
来自自己和别人的怨气已经侵染入血，不能改变自己的忧郁。


自己本早已心灰意冷，不知被随手之画伤害到的你。
假使当初你什么都没说，我大可以如平日里，玩世不恭依然，呵呵哈哈依然。
但是你说出了我听到了就要承担的东西。你知道，自己尚且难以忍受的痛楚，
嫁接给任何一个人都会是更大的伤害。
当我走在这个满树乌鸦的园子里，已经不能欣喜于落在肩膀的鸟粪，
因为，我是那个唯一，唯一知道，在那个窗台，
趴着那只我的猫，但是我看不到。

给我发电子邮件吧，我不可能知道你是谁的。
只要知道你好我就一切都好了。

还有3幅画我会保存，直到也许只能是永远的那一刻

windowsxp／linux双系统引导管理器GRUB修复

windowsxp／linux双系统引导管理器GRUB修复作者:lax 日期:2007-01-04

电脑上一直使用双系统，winxp和fedora core 6，使用grub引导器。
昨天不小心把win删除了，无聊之下重装。装好之后才想起没有备份grub，于是没有办法进入我可爱的linux了。

于是查文档。网上提到的有用WinGRUB或者GRUB for DOS引导的东东，但是实在是不过确认能否有效。还是最初的fc6安装盘比较保险。于是将fc6boot.iso的那个盘放了进去。F5后输入linux rescue进入救援模式；简单点下一步和选择安装文件的位置之后（我的安装文件都放在hda2下sys文件夹里面了;-)进入命令行操作模式。这个程序ana＊＊＊做得比较好，将硬盘上存在的系统直接映射到/mnt/sysimage目录了，于是cat /mnt/sysimage/boot/grub/grub.conf，从里面可以看到要root位置(hd0,0)，和kernel的版本，记录下来。

接着运行grub，
root (hd0,0) 回车
kernel /vmlinuz-2.*****(太长了，没记下，可以按tab键自动补全，这是linux一个很好的功能) 回车
initrd /initrd-2.****(同上) 回车

以上配置好了，下面安装grub到硬盘MBR
setup (hd0) 回车
提示成功，
重启, OK.

古北口之行记略

12月13日中午乘火车出发，三点半之后就来到了北京河北的交界处古北口。

最能刺激人视网膜细胞的莫过于这里的山，和巍巍山巅间遥相呼应的烽火台。

第一次感受到人在自然中只是如此渺小的存在。

因为平原之处出生的我只见到被人类驯养的服服帖帖的农田，偶尔见到被人类过度使用的河流，总会为人类的铁手腕而叹息。

但在这里，再也找不到叹息的理由，因为人类根本就没有能力把这个大自然驯服，哪怕是简单合作也要费尽心机。

在山下的古镇古北口简单的休整之后，我们四个尝试登上长城。

天气开始变冷，也慢慢开始黑下来了，登上顶峰眼看是不可能完成了。

更郁闷的是，通过铁路之后我们走了另外一条道，可以通向高处的烽火台，而走了一多半我们才发现时间已经不允许了，这时候同时出发的几个小伙子已经在山对面较低的烽火台上跟我们对话并准备下手了。

天渐渐黑下来，终于，一条悬崖小道让我们丧失了继续向上的信心。

我们在的这个地点视野很开阔，可以看见整个村庄在山谷中，铁路在脚下只是很小的一条线，穿过几个隧洞。

往东看大概就是河北的地域了。长城从远处过来，绵延不断。

很安静的感觉，彼此说话的声音是如此清晰，毫不费力。

好久之后我们决定下山，沙石路有点滑，我们就手挽手。

上山容易下山难，但是没有想象中的难。

织女星出现了，又有几个陆续露面。

下山的时候走错一段路，突然冒出一个很绝望的想法：如果真的在这样的黑夜这样的山路迷失了，后果呢？没有去想。好像没有这个必要，或者是已经麻木了。
